Simple, fast and resource-intensive script for PC and Aruduino. 

Made to create a Display with the CPU and GPU temperatures and also with control of the fan speeds by PWM, 
adding extra outputs to your system, ensuring silence and good cooling. 

All the use is by directly editing the files (A graphic interface for this is in the near plans) 
but even so due to the simplicity of the project this does not become something difficult even for beginners. 

Some knowledge in electronics is always good, Help with Arduino programming and connections. 

You need to have the Arduino IDE, Python, and the psutil, serial, and time dependencies. 

This project for now is only available and working on Linux, windows support will be added in the distant future ;)
